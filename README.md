# Speed Typing

Type a full paragraph as fast as possible with WPM and accuracy tracking.

## How to Use

**Step 1:** Open `index.html` in your browser.

**Step 2:** An onboarding tutorial appears on first visit. Click **Start Typing** (or Skip).

**Step 3:** Read the paragraph displayed above the input area.

**Step 4:** Start typing in the textarea. The timer begins automatically when you type.

**Step 5:** Green text = correct characters. Red text = mistakes.

**Step 6:** Monitor your WPM (Words Per Minute), Accuracy, and Time in real-time.

**Step 7:** When you finish the entire paragraph, your final results appear.

**Step 8:** Switch difficulty: **Easy** (simple sentences), **Medium** (tech topics), **Hard** (complex text).

## Features
- 3 difficulty levels
- Real-time WPM calculation
- Accuracy percentage tracking
- Character-by-character color feedback
- Final results summary
- Timer with millisecond precision
